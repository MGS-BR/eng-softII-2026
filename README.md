# 📊 Contabilidade Ágil

[![PlantUML](https://github.com/Engenharia-de-Software-Fatec/eng-softII-2026/actions/workflows/plantuml.yml/badge.svg)](https://github.com/Engenharia-de-Software-Fatec/eng-softII-2026/actions)

Projeto desenvolvido para a disciplina de **Engenharia de Software II** do curso de **Análise e Desenvolvimento de Sistemas (ADS)** da Fatec Sorocaba.

---

## 🚀 Sobre o Projeto

O **Contabilidade Ágil** é uma aplicação com dashboard que automatiza lançamentos contábeis por meio de Inteligência Artificial.

A proposta é permitir que empresas enviem documentos contábeis e que o sistema:

* 📥 Receba os documentos do cliente
* 🤖 Utilize IA para gerar lançamentos contábeis automaticamente
* 📊 Organize os lançamentos nas contas corretas
* 👨‍💼 Permita que o contador revise, aprove ou corrija os dados

---

## 🎯 Objetivo

Desenvolver uma aplicação inteligente que aumente a produtividade de escritórios contábeis, reduzindo erros manuais e otimizando o processo de escrituração.

---

## 💰 Custos

* **Custo sem a aplicação:** ≈ R$ 109,58 p/documento

---

## 👥 Atores do Sistema

* **Cliente (Empresa)**
  Responsável por enviar os documentos contábeis.

* **Contador (Administrador)**
  Responsável por validar, corrigir e aprovar os lançamentos.

* **Sistema de IA**
  Responsável por interpretar os documentos e gerar os lançamentos automaticamente.

---

## ⚠️ Principais Dores

* Processos manuais demorados
* Alto risco de erro humano
* Grande volume de documentos
* Baixa produtividade em escritórios contábeis
* Dificuldade de padronização dos lançamentos
